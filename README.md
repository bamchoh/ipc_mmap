# ipc_mmap

This library is IPC(interprocess communication) over mmap and POSIX semaphore.
This library was implemented by refering the following blog:
[mmap＋POSIXセマフォによる高速なプロセス間通信 - mp::ipc_vector](http://frsyuki.hatenablog.com/entry/20081028/p1)

# How to build a library

libclient.a and libserver.a will be generated by the following command.
You have to install ruby in your system before executing the command.

```sh
# rake all
```

# How to exmaple binary tools

client and server will be generated by the following command.

```sh
# rake example
```
